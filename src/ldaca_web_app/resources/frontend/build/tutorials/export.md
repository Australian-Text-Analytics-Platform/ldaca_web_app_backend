<h1 id="help-export-section">Export tutorial</h1>

![Export screenshot](tutorials/assets/export.png)

Export lets you download tables or results for reports, sharing, or additional analysis in other tools.

> **Placeholder (GIF):** Exporting results to CSV.

<h2 id="help-export-parameters">Parameter panel</h2>

Use the parameter panel to choose the export format and preview details.

<h3 id="help-export-format">Export format selector</h3>

Choose the file format for your export (CSV, Excel, etc.). Pick the format that matches your downstream workflow.

**Q: Which format should I choose?**

Use **CSV** for maximum compatibility, and **Excel** if you need multiple sheets or formatted output.

<h2 id="help-export-results">Result panel</h2>

Use the result panel to generate and download the export.

<h3 id="help-export-run">Export action</h3>

Click the export action to generate the file. Once complete, a download link appears.
