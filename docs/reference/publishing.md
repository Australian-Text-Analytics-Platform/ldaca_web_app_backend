# Backend Publishing (PyPI)

**Scope statement:** Publish the backend package for `uvx`‑style usage.

## Step 1 — Build the package

**Question:** *How do I build the backend distribution?*

**Answer:** Use the workspace build command from the repository root or backend directory.

## Step 2 — Upload

**Question:** *Where do I upload the artifacts?*

**Answer:** Upload to TestPyPI first, then PyPI once validated.

## Step 3 — Verify

**Question:** *How do I verify the release?*

**Answer:** Run the CLI via `uvx` or install the wheel in a clean environment.

## Recap

**Question:** *Where are the exact commands?*

**Answer:** Use the backend packaging scripts and the publishing checklist in this repository’s release process.
