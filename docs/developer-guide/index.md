# Backend Developer Guide

This guide documents how the FastAPI backend is implemented.

- [Architecture](architecture.md): big-picture backend responsibilities.
- [Application lifecycle](application-lifecycle.md): startup, settings,
  database, auth, frontend mounting, and CLI server launch.
- [Workspace and files](workspace-and-files.md): user folders, workspace
  manager, file APIs, node APIs, and schema filtering.
- [Analysis and workers](analysis-and-workers.md): task stores, worker process
  pool, SSE events, analysis modules, caches, and materialization.
- [API patterns](api-patterns.md): router conventions, response shapes,
  authentication, and implementation rules.

For user-facing API summaries, see `../reference/`.
